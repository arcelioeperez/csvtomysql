# CSV to MySQL  

### How to use the `tomysql` executable:  

```
git clone https://github.com/arcelioeperez/csvtomysql.git && cd csvtomysql
chmod +x tomysql 
./tomysql
```  

**Alternatively run with**  

```
python3 main.py
```  

### Sample case:  

-The prompts will appear on the terminal.  

### Requirements:    
pandas==1.1.5   
SQLAlchemy==1.3.21  
PyMySQL==0.10.1
